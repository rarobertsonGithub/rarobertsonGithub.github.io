Creates a GUI to allow use of the implemented functionality outside of the terminal, allows users to select a text file to encode using huffman encoding, encodes the files and stores both the encoded files and the actual huffman codes, allows users to take both the huffman codes & the encoded text file & decode that text file into the original human readable text, and displays data about the ratio of the compressed file to the uncompressed file.
