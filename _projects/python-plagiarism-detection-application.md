Creates a GUI to allow use of the implemented functionality outside of the terminal, allows users to select folders in which all of the files in said folder will be displayed for in application selection, allows for sorting of said folders files based on different criteria (Title, Author, Creation Date), allows user to select the files that will be compared/analyzed for instances of plagiarism, creates a json log with custom formatting so that all instances of plagiarism may be saved for later use, parses said logs and uses that data to create a visualization in a tree structure that displays the portions of text that have been plagiarized from the other selected text file.
